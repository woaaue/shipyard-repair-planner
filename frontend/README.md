# Док-План

Система планирования судоремонта на базе React + TypeScript + Vite.

## Функционал

- Управление судами и ремонтами
- Календарное планирование и диаграмма Ганта
- Ролевая система доступа (admin, operator, master, worker, client)
- Управление пользователями и доками
- Отчёты и экспорт данных
- Глобальный поиск

## Технологии

- React 19 + TypeScript
- Vite 7
- Tailwind CSS
- Recharts (графики)
- React Router DOM 7
- Vitest (тестирование)

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Тесты

```bash
npm run test
```

## Роли пользователей

| Роль | Описание |
|---|---|
| Администратор | Полный доступ к системе |
| Оператор дока | Управление конкретным доком |
| Мастер участка | Контроль ремонтных работ |
| Рабочий | Просмотр и выполнение задач |
| Владелец судна | Просмотр статуса своих судов |

## Структура проекта

```
src/
├── components/     # UI компоненты
├── context/        # React Context (Auth, Dock, Repair, Toast)
├── hooks/          # Кастомные хуки
├── mock-data/      # Моковые данные
├── pages/          # Страницы приложения
├── services/       # API-слой
├── types/          # TypeScript типы
└── utils/          # Утилиты
```
